Introduction

This application is developed to create spend analysis of the bank statements uploaded by the users :

Register- User can register themselves on the site and set their password.

Login- User can login into the system with the registered credentials.

Manage Bank Accounts- User can add / removed their bank accounts in the database.

Upload Statement- Users can select their bank account and upload their statements in excel format into the system.

Spend Analysis- Users can do the spend analysis on the uploaded statement for a specific account and a date period.

Add Bank Details (Admin Only)- Admin users can maintain the Bank names and the format of excel statement to assist the system in uploading the data into the system.
